# Testing Folder

## Purpose
This folder contains introductory examples and exercises designed to help new team members get familiar with GitHub and Python/MATLAB, which may be unfamiliar systems for those joining the UAS Hyperspectral team. Feel free to edit these files to test pushing and pulling changes in Github.

## Contents

- **Github_Test.m & Read_Numbers.m** - Simple MATLAB scripts demonstrating basic outputs and functions meant to help familiarize team members with the Github-MATLAB inferface. A good starting point for new members to the team unfamiliar with Github.

- **pythonTest.py** - A minimal Python script showing a basic `print()` statement. A good introduction for team members learning Python and navigating the Github interface in VSCode.

- **Numbers.txt** and **TextTest.txt** - Sample data files used by the MATLAB scripts for practice.

## Getting Started
1. Start with `Github_Test.m` or `pythonTest.py` depending on your primary language
2. Explore `Read_Numbers.m` to see how to work with data files
3. These examples build familiarity with both GitHub repositories and basic programming workflows

## Next Steps
Once comfortable with these examples, team members are ready to explore the main project files and contribute to the UAS Hyperspectral project.
